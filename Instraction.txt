**Activity / Fragment Transition Animation (Fade) – Description**

Definition:
Transition animation হলো একটি visual effect যা ব্যবহারকারীর জন্য smooth screen change experience তৈরি করে। যখন একটি screen (Activity বা Fragment) আসছে বা যাচ্ছে, তখন animation ব্যবহার করলে UI আরও সুন্দর এবং natural লাগে।

Key Points

1. Activity Level Animation

   * যখন এক Activity থেকে অন্য Activity-তে যাওয়া হয়।
   * ব্যবহার: overridePendingTransition(int enterAnim, int exitAnim)
   * উদাহরণ:
     startActivity(new Intent(this, BActivity.class));
     overridePendingTransition(R.anim.fade\_in, R.anim.fade\_out);
   * Enter animation → নতুন Activity আসার সময়
   * Exit animation → পুরানো Activity যাওয়ার সময়

2. Fragment Level Animation

   * যখন এক Fragment থেকে অন্য Fragment replace বা add হয়।
   * ব্যবহার: FragmentTransaction.setCustomAnimations(int enter, int exit, int popEnter, int popExit)
   * উদাহরণ:
     fragmentTransaction.setCustomAnimations(
     R.anim.fade\_in,  // নতুন fragment আসবে
     R.anim.fade\_out, // পুরানো fragment যাবে
     R.anim.fade\_in,  // back চাপলে আসবে
     R.anim.fade\_out  // back চাপলে যাবে
     );
     fragmentTransaction.replace(R.id.frameLayout, newFragment);
     fragmentTransaction.commit();
   * PopEnter → back চাপলে fragment আসার সময়
   * PopExit → back চাপলে fragment যাওয়ার সময়

3. Why Fade is Useful

   * Smooth transition → UI look & feel improve হয়
   * User confusion কমায় → কোন screen চলে গেছে বা এসেছে বোঝা যায়
   * Professional apps (Instagram, TikTok, Facebook) সবেই animation ব্যবহার করে

4. Important Note

   * Activity → Activity animation overridePendingTransition() শুধুমাত্র startActivity() বা finish() এর পর কাজ করে।
   * Fragment → Fragment animation setCustomAnimations() FragmentTransaction এর মধ্যে কাজ করে।
   * Back button press → FragmentAnimation ব্যবহার করতে হলে FragmentTransaction এ PopEnter & PopExit set করতে হবে।
   * Hardware / system back button → Activity level animation চাইলে onBackPressed() override করতে হয়।

Summary:
Fade Transition Animation হলো একটি screen change effect যা Activity বা Fragment replace / start / back press এর সময় smooth দেখায়।
Professional apps সবসময় screen transition effect ব্যবহার করে UX improve করার জন্য।
